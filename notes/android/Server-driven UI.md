## Server-driven UI

- 서버에서 도메인 데이터뿐만 아니라 레이아웃 정보도 함께 제공
- 레이아웃 코드를 수정하고 구글 플레이에 배포한 뒤 리뷰를 기다리고, 
  사용자가 업데이트하기까지의 배포 프로세스를 거치지 않아도 됨
- JSON 형식으로 UI 데이터를 전송, 프론트에서 이를 변환하여 처리
